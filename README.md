# SpendObserver

A local-first expense tracker focused on privacy, flexibility, and full control over personal financial data.

## Features

### Financial Records
Create, update, and delete income and expense records
Records are organized by years and months

Each record supports:
- custom tags
- optional text description
- a built-in calculator to compute amounts directly in the input field (useful for splitting or merging receipts)

### Multiple Trackers
Create and manage multiple independent trackers
Suitable for separating personal, family, or project-related expenses

### Tag Management
Add, edit, and delete tags
Fully customizable categorization system

### Import / Export
- Save trackers as files
- Load existing trackers from files
- Share trackers between devices
